Características Principais:

• Linguagem OO (Orientada a Objeto) / Funcional
• Interoperabilidade com Java
• Kotlin + Java juntas
• Sintaxe enxuta e intuitiva
• Curva de aprendizado simplificada
• Criada pela JetBrains

História

1990:

• A linguagem Java é criada pela empresa Sun Microsystems

2009:

• Sun é comprada pela Oracle
• Linguagem Java e a JVM (Java Virtual Machine) se tornam propriedade da Oracle
• Java era open-source e algumas partes da JVM fechadas

2010:

• Java é escolhida pelo Google como linguagem padrão de desenvolvimento para o sistema Android
• Android não usa a JVM, mas uma máquina virtual desenvolvida pelo Google chamada Dalvik
• Dalvik compreende o bytecode gerado pelo Java
• Oracle alega que Google teria copiado trechos dos códigos dela, infringindo patentes de 37 APIs da JVM dentro da Dalvik
• Oracle processa Google baseada na alegação acima
• Após este episódio, as relações entre Google e Oracle não ficaram boas
• Dalvik deixou de existir e foi substituída pela ART (Android Runtime)
• Desde então, o Google começou uma corrida silenciosa e não aberta em busca de uma linguagem para substituir o Java no ambiente Android

2011:

• JetBrains começa o desenvolvimento da linguagem Kotlin
• A linguagem foi nomeada em homenagem a uma ilha localizada no Golfo da Finlândia, na Rússia, região onde fica a cidade de São Petesburgo, localização de um dos principais escritórios da JetBrains
• O líder de desenvolvimento do projeto, Hadi Hariri, desenvolve a linguagem Kotlin
• JetBrains cria a linguagem Kotlin com a finalidade de facilitar a criação de suas IDEs, visto que o Java dava muito trabalho para desenvolver estas IDEs
• A empresa queria uma linguagem que funcionasse parecido com o Java, porém escrevendo menos

2012:

• Um ano após sua criação, a linguagem Kotlin já se torna uma linguagem open-source

2016:

• Surge o Kotlin 1.0

2017:

• No final do evento Google I/O, o Google anuncia a linguagem Kotlin como mais uma linguagem para o desenvolvimento oficial de aplicações para Android
• Google entra oficialmente como um dos grandes parceiros da JetBrains para integrar a linguagem Kotlin ao Android Studio
